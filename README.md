# 🧠 Temporal Deepfake Detection Pipeline

A deep learning system for detecting synthetic facial manipulation in video using a **spatial-temporal hybrid architecture**: **EfficientNetV2-S** (spatial feature extraction) + **Bidirectional LSTM** (temporal consistency analysis).

---

## 📊 Dataset Preparation

### Option A — FaceForensics++
1. Request access at [github.com/ondyari/FaceForensics](https://github.com/ondyari/FaceForensics)
2. Download and extract to `data/FaceForensics/`
3. Generate the unified CSV:
```python
from data_loader import scaffold_ff_plus_plus
scaffold_ff_plus_plus()
```

### Option B — DFDC
1. Download from [Kaggle DFDC](https://www.kaggle.com/c/deepfake-detection-challenge/data)
2. Extract to `data/DFDC/`
3. Generate the unified CSV:
```python
from data_loader import scaffold_dfdc
scaffold_dfdc()
```

## 🚀 Training

```powershell
# Standard training
python train.py

# Pipeline smoke test (no dataset required — uses synthetic data)
python train.py --epochs 2 --batch_size 2 --smoke_test

# Resume from last checkpoint
python train.py --resume outputs/checkpoints/last_model.pth

# Fine-tune with unfrozen backbone
python train.py --freeze_backbone false --lr 1e-5
```

## 🏗️ Architecture

```
Input: (batch, seq_len=20, 3, 224, 224)
         ↓
 SpatialEncoder (EfficientNetV2-S, ImageNet pretrained)
   → per-frame 1280-d feature vector
         ↓  reshape → (batch, seq_len, 1280)
 TemporalClassifier (2-layer Bidirectional LSTM)
   → mean-pool → FC(1024→256) → GELU → Dropout → FC(256→1)
         ↓
Output: (batch, 1) — raw logit → σ → fake probability
```

## 🔧 Key Configuration (`config.py`)

| Parameter | Default | Description |
|---|---|---|
| `FRAME_STEP` | 5 | Extract every N-th frame |
| `SEQ_LEN` | 20 | Frames per sequence window |
| `IMG_SIZE` | 224 | Face crop resolution |
| `BATCH_SIZE` | 4 | Training batch size |
| `EPOCHS` | 30 | Max training epochs |
| `EARLY_STOP_PAT` | 5 | Early stopping patience |
| `FREEZE_BACKBONE` | True | Freeze EfficientNetV2 initially |
